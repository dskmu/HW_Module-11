# UML Диаграммы — Draw.io

## Задание 1: Электронный магазин
**Тип:** UML диаграмма классов

Разработана UML-диаграмма классов для системы электронной коммерции.

### Включает:
- Модуль управления пользователями (User, Client, Admin)
- Модуль управления товарами (Product, Category)
- Модуль оформления заказов (Order, OrderItem, Cart, PromoCode)
- Модуль финансов (Payment, CardPayment, WalletPayment)
- Модуль логистики (Delivery, Courier)
- Модуль отзывов (Review)

### Инструмент: Draw.io

## Задание 2: Система логистики и доставки
**Тип:** UML диаграмма компонентов

Разработана UML-диаграмма компонентов для системы управления логистикой и доставкой товаров.

### Включает:
- Frontend (веб-приложение, мобильное приложение)
- Backend (API Gateway, управление заказами, аутентификация)
- База данных
- Модуль оптимизации маршрутов
- Модуль управления складом
- Система уведомлений
- Система аналитики
- Внешние системы (курьерские службы, платёжные системы, карты)

### Инструмент: Draw.io
